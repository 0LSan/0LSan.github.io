软件介绍：

Microsoft Office是 由Microsoft(微软)公司开发的一套基于一套基于 Windows 操作系统的办公软件套装。常用组件有 Word、Excel、PowerPoint、Outlook等。

安装步骤：

  1. 鼠标右击【Office2019】压缩包选择【解压到 Office2019】。
  2. 打开解压后的【Offce2019】文件夹，鼠标右击【ProPlus2019Retail】选择【装载】。
  3. 打开装载后的文件夹，鼠标右击【Setup】选择【以管理员身份运行】。
  4. 软件安装中（大约需要2分钟），安装结束后会自动关闭安装界面。
  5. 打开安装包解压后的【Office2019】文件夹，双击打开【激活工具】文件夹。
  6. Office激活步骤：

    7. 方法一：

    温馨提示：如果激活工具文件夹里没有“MicroKMS”或“打不开此程序”，请关闭电脑上的“杀毒软件、防火墙”后重新解压安装包，Win10系统还需关闭“Windows Defender”。

    1. 点击【激活Office 2010– 2019 ProPlus】。
    2. 激活成功，点击【确定】。

    方法二：

    1. 鼠标右击【KMS-VL-ALL_cn】选择【以管理员身份运行】。
    2. 在弹出的界面等待一下，自动激活。

      3. 温馨提示：方法二不是永久激活，当方法一激活失败时可尝试用方法二，激活成功使用期限是180天，到期后按照以上方法重新激活即可。

    3. 安装成功。
  7. 在桌面空白处，鼠标右键选择【新建】选择【Microsoft Word 文档】。
  8. 双击打开【新建 Microsoft Word 文档】。
  9. 点击【接受并启动Word】。
  10. 激活完成。
